### What is RondoFlow?

**RondoFlow is a local-first, open-source visual orchestration platform for building and running multi-agent Claude Code workflows on a drag-and-drop canvas.** Connect agents, skills, security policies, and resources, then execute multi-agent runs entirely on your own machine.

Claude Code is a single agent in your terminal. RondoFlow turns it into a team. Drop agents, skills, and guardrails onto a board, connect them, and **the graph you draw is the workflow that runs** - branching across agents in parallel, with each step shown live on the canvas.

> A Planner tunes the workflow before the run, a Director steers each step during the run, and an Advisor reviews the result afterward - all on your own machine.

### Why teams choose RondoFlow

- 🎨 **Draw it, don't code it.** Drag agents, skills, and guardrails onto a visual canvas and connect them. No YAML, no boilerplate - the board you build is the pipeline that executes.
- 👥 **Many agents, in parallel.** Run a whole crew at once - reviewers, researchers, writers - instead of one terminal session, one turn at a time. RondoFlow walks the DAG, branches the work, merges results, and shows each step live.
- 🔒 **Local-first and private by default.** Everything runs on your machine against the Claude Code CLI. Your code, prompts, and keys never leave it - only the messages you send to a model reach that model's provider.
- 🧠 **AI that plans, steers, and reviews.** A Planner tunes before the run, a Director steers mid-run, and an Advisor reviews after - so your agent runs get smarter without your babysitting them.
- 🛡️ **Per-agent guardrails.** Give every agent its own security policy - allowed tools, paths, and permissions. When rules overlap, the **most restrictive one always wins**, so adding a rule can only tighten behavior.
- ⏰ **Set it and forget it.** Put any workflow on a cron schedule and let it run on its own - reports, reviews, and research while you are away.
- 🔋 **Batteries included.** Start from a shipped catalog of ready-to-use agents, skills, workspaces, facilitators, and canvas templates - then tweak them to fit.
